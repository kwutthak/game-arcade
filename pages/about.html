About This Website

This little website intends to be a collection of a few simple arcade game selections where you can play straight from your browser! The first version of this website includes 3 games in total with more potentially being added some time in the future. As you can already tell, the theme of this website is heavily inspired by sanrio creations with soft aesthetics and cute cartoon characters. My intended audience for this website is children to teenagers as the games tend to be more on a simpler side but nevertheless everyone is welcome. The game collection currently includes:

  • Whack-A-Mole
  • Memory Cards
  • Dino Runner
